Lyuba!!!

  • assets - contains static files (fonts, images, documents, swf etc.)
  • dist
    • src - contains full source code without third-party settings (like bower or grunt)
    • debug - contains the website with LESS files compiled to CSS
    • release - contains compiled and minified version of the project, ready to upload
  • features - contains features for cucumber tests
    • step_definitions - contains steps definitions for cucumber tests
    • support - contains support files which let you setup the environment in which steps will be run
  • libs - contains third-party components, managed by Bower
  • scripts - contains all custom Javascript code
  • styles
    • base - contains common styles for the whole application
    • controls - contains styles for reusable components
    • pages - contains styles for specific pages
    • utils - contains useful mixins and utilities
